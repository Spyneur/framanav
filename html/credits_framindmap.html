Framindmap repose sur le logciel libre Wisemapping.

Ce logiciel est compatible avec les navigateurs web suivant :

  • Microsoft Internet Explorer 9+
  • Google Chrome 19+
  • Firefox 12+
  • Safari 5+
  • Opera 11+

Il est sous licence WPL.

Auparavant, Framindmap reposait sur le logiciel Mindmaps de David Richard.
Ce logiciel dispose d'une interface plus simple et ne nécessite ni de créer un compte ni de faire transiter vos données sur notre serveur.
