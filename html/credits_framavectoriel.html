Pour toute question concernant Framavectoriel, vous pouvez envoyer utiliser le site http://contact.framasoft.org

Framavectoriel est une application permettant de réalisser des dessins au format vectoriel (SVG).
Il s'agit de la version française, traduite par l'association Framasoft, de l'application "svg-edit"

Framvectoriel est placé sous licence libre Creative Commons BY-SA 3.0

La page d'accueil utilise le framework libre Boostrap ainsi que la librairie jQuery.
